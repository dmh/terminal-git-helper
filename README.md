# Terminal-git-helper

####Manual installation:

1 Clone the repo in your user directory.

2 Then add `source ~/user/terminal-git-helper/terminal_git_helper.sh` to the bash startup file (`~/.bash_profile` or `~/.profile` on Mac)


####Tested only on Mac!
